    In `NodeEditorScreen.kt`, add 4 new imports at the top (after the existing imports, around line 81):
    ```kotlin
    import androidx.compose.runtime.DisposableEffect
    import androidx.lifecycle.Lifecycle
    import androidx.lifecycle.LifecycleEventObserver
    import androidx.lifecycle.compose.LocalLifecycleOwner
    ```

    CRITICAL: Use `androidx.lifecycle.compose.LocalLifecycleOwner` (NOT `androidx.compose.ui.platform.LocalLifecycleOwner` which is deprecated in Compose 1.8+).

    Add the DisposableEffect inside the `NodeEditorScreen` composable, after the `LaunchedEffect(Unit)` sideEffect collector block (after line 116, before the `when (state.status)` block):
    ```kotlin
    // GAP-V2: Wire lifecycle to sync callbacks
    val lifecycleOwner = LocalLifecycleOwner.current
    DisposableEffect(lifecycleOwner) {
        val observer = LifecycleEventObserver { _, event ->
            if (event == Lifecycle.Event.ON_RESUME) viewModel.onScreenResumed()
            if (event == Lifecycle.Event.ON_PAUSE) viewModel.onScreenPaused()
        }
        lifecycleOwner.lifecycle.addObserver(observer)
        onDispose { lifecycleOwner.lifecycle.removeObserver(observer) }
    }
    ```
