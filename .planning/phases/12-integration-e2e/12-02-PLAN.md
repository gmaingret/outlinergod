**AppRoutes.kt** — The NODE_EDITOR route currently is `"node_editor/{documentId}"`. It needs an optional `rootNodeId` query parameter for zoom. Do NOT change the route pattern itself (that would break existing navigation). Instead, add a helper function:

```kotlin
object AppRoutes {
    const val LOGIN = "login"
    const val DOCUMENT_LIST = "document_list"
    const val DOCUMENT_DETAIL = "document_detail/{documentId}"
    const val NODE_EDITOR = "node_editor/{documentId}?rootNodeId={rootNodeId}"
    const val SETTINGS = "settings"
    const val BOOKMARKS = "bookmarks"

    fun nodeEditor(documentId: String, rootNodeId: String? = null): String {
        val base = "node_editor/$documentId"
        return if (rootNodeId != null) "$base?rootNodeId=$rootNodeId" else base
    }
}
```

IMPORTANT: The route constant with the query param template is needed for Compose Navigation to recognize the optional argument. The `nodeEditor()` function builds actual navigation URLs.
