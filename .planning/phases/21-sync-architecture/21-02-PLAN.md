SyncWorker.kt (removes NodeDao, DocumentDao, BookmarkDao, SettingsDao, DataStore injections):
```kotlin
@HiltWorker
class SyncWorker @AssistedInject constructor(
    @Assisted appContext: Context,
    @Assisted workerParams: WorkerParameters,
    private val syncOrchestrator: SyncOrchestrator,
    private val authRepository: AuthRepository   // keep for any non-sync auth needs; verify if still needed
) : CoroutineWorker(appContext, workerParams) {

    override suspend fun doWork(): Result {
        return if (syncOrchestrator.fullSync().isSuccess) {
            Result.success()
        } else {
            Result.retry()
        }
    }
}
// NOTE: authRepository is only needed if SyncWorker has other uses for it beyond sync delegation.
// Per CONTEXT.md, SyncWorker removes ALL DAO injections. Verify authRepository is still needed
// (current doWork() uses it only for refreshToken — now absorbed into orchestrator). If authRepository
// has no remaining use in SyncWorker after the refactor, remove it from the constructor too.
// SAFETY: removing authRepository from SyncWorker constructor is only valid if no other method in
// SyncWorker uses it. Confirm by reading the full current SyncWorker.kt before editing.
```
