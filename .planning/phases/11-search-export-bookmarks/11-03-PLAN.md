**1. Create `ui/screen/bookmarks/BookmarkListViewModel.kt`:**
- `@HiltViewModel class BookmarkListViewModel @Inject constructor(bookmarkDao: BookmarkDao, authRepository: AuthRepository, hlcClock: HlcClock)`
- Extends `ViewModel(), ContainerHost<BookmarkListUiState, BookmarkListSideEffect>`
- `sealed class BookmarkListUiState`:
  - `data object Loading : BookmarkListUiState()`
  - `data class Success(val bookmarks: List<BookmarkEntity> = emptyList()) : BookmarkListUiState()`
  - `data object Empty : BookmarkListUiState()`
- `sealed class BookmarkListSideEffect`:
  - `data class NavigateToDocument(val documentId: String) : BookmarkListSideEffect()`
  - `data class NavigateToNodeEditor(val documentId: String) : BookmarkListSideEffect()`
  - `data class ShowError(val message: String) : BookmarkListSideEffect()`
- **Container declaration with explicit initial state:** `override val container = container<BookmarkListUiState, BookmarkListSideEffect>(BookmarkListUiState.Loading)`
- `init { loadBookmarks() }`
- `fun loadBookmarks() = intent {}`:
  - `val userId = authRepository.getUserId().filterNotNull().first()`
  - Collect `bookmarkDao.observeAllActive(userId)` and reduce to Success or Empty based on list emptiness
- `fun onBookmarkTapped(bookmark: BookmarkEntity) = intent {}`:
  - When `targetType == "document"`: `postSideEffect(NavigateToDocument(bookmark.targetDocumentId!!))`
  - When `targetType == "node"`: `postSideEffect(NavigateToNodeEditor(bookmark.targetDocumentId!!))` — navigate to the document containing the node
  - When `targetType == "search"`: for now, just log or no-op (search navigation will be wired in Plan 04 if needed)
- `fun softDeleteBookmark(id: String) = intent {}`:
  - Get deviceId, generate HLC, call `bookmarkDao.softDeleteBookmark(id, now, hlc, now)`
  - Follow exact pattern from DocumentListViewModel.deleteDocument()

  - Swipe-to-delete or trailing IconButton for delete
  - On click: call `viewModel.onBookmarkTapped(bookmark)`
- Collect side effects: NavigateToDocument -> onNavigateToDocument(docId), NavigateToNodeEditor -> onNavigateToNodeEditor(docId), ShowError -> show Snackbar
